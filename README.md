# 🧮 CLI Calculator (Python)

A simple command-line calculator built using Python.  
It performs basic arithmetic operations directly from the terminal.

---

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponent/Power (**)
- Modulus/Remainder (%)
- Continuous usage in terminal
- Error handling (division by zero, invalid input)
- Beginner-friendly CLI interface

---

## 🛠️ Tech Stack

- Python 3.14.4
- Standard Python libraries (no external dependencies)

---

## 📦 Installation

1. Clone the repository:
git clone https://github.com/Forte-Romeo/cli-calculator-python.git

2. Navigate into the project folder:
cd cli-calculator-python

3. Run the program:
python calculator.py

---

## ▶️ How to Use

When you run the program, you will be prompted to:
- Choose an operator choice (+, -, *, /, **, %)
     - Addition (#1)
     - Subtraction (#2)
     - Multiplication (#3)
     - Division (#4)
     - Exponent (#5)
     - Modulus (#6)
- Enter the first number
- Enter the second number

Example:

- Enter operator choice (eg. Multiplication): 3
- Enter first number: 10
- Enter second number: 5
- Result: 50

---

## ⚠️ Error Handling

This project handles:
- Division by zero
- Invalid operators
- Non-numeric inputs
