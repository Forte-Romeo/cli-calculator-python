# 🧮 CLI Calculator (Python)

A simple command-line calculator built using Python.  
This project performs basic arithmetic operations directly from the terminal while demonstrating core Python programming concepts such as functions, loops, conditionals, input validation, and error handling.

---

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponent/Power (**)
- Modulus/Remainder (%)
- Continuous calculations without restarting the program
- Error handling (division by zero, invalid input)
- Beginner-friendly CLI interface

---

## 🛠️ Tech Stack

- Python 3.14.4
- Standard Python libraries (no external dependencies)

---

## 📦 Installation

1. Clone the repository:
`git clone https://github.com/Forte-Romeo/cli-calculator-python.git`

2. Navigate into the project folder:
`cd cli-calculator-python`

3. Run the program:
`python calculator.py`

---

## ▶️ How to Use

When you run the program, you will be prompted to:

- Select a mathematical operation
     - Addition (#1)
     - Subtraction (#2)
     - Multiplication (#3)
     - Division (#4)
     - Exponent (#5)
     - Modulus (#6)
- Enter the first number
- Enter the second number
- View the result
- Continue calculating using the previous result or return to the main menu

Example:

```
===== CLI CALCULATOR ===== 

1. Add
2. 2. Subtract
3. Multiply
4. Divide
5. Exponent
6. Modulus
7. Exit

Enter your choice: 3
Enter first number: 10
Enter second number: 5

Result: 50
```

---

## 🔄 Continuous Calculation Feature

After each calculation, the program allows you to continue calculating using the previous result.

Example:

```
10 + 5 = 15
15 * 2 = 30
30 - 10 = 20
```

This creates a smoother calculator experience similar to a real calculator.

---

## ⚠️ Error Handling

This project handles:
- Division by zero
- Invalid menu choices
- Non-numeric inputs
- Invalid calculations
